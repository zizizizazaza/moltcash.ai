# Loka 信用分系统 (Credit System)
项目方与投资者的完整激励与治理方案

## 1. 🏢 项目方信用分系统 (Enterprise / Issuers)

**核心设计**：必须先完成认证才能募资，不能"先上车后补票"。

### 1.1 初始状态与门槛
| 项目 | 数值 | 说明 |
| --- | --- | --- |
| 初始信用分 | 100 分 | 刚注册，仅完成钱包连接 |
| 募资门槛 | 200 分 | 必须达到才能发起募资 |
| 初始等级 | 待认证 | 无法募资 |

### 1.2 认证解锁 (+100分)
| 认证步骤 | 具体内容 | 得分 |
| --- | --- | --- |
| 企业信息填报 | 注册名称、国家/地区、注册号、注册地址 | +25 |
| 核心证照上传 | 营业执照、近3个月水电单/银行对账单 | +25 |
| 股权结构披露 | 声明持有25%以上股份的所有股东身份 | +25 |
| 实际控制人KYC | 核心股东+法人代表：护照/身份证+人脸识别 | +25 |
| **合计** | - | **+100** |

### 1.3 三档权益体系
信用分直接挂钩融资的“抵押率”和“平台费”。

| 档位 | 信用分要求 | 抵押率 | 平台费 | 等级名称 |
| --- | --- | --- | --- | --- |
| 一档 | 200 分 | 30% | 5% | 新秀 |
| 二档 | 500 分 | 10% | 3% | 成长 |
| 三档 | 1000 分 | 10% | 1% | 蓝筹 |

### 1.4 分数获取途径
#### 行为 1：按时还款（核心来源）
| 行为 | 得分 | 说明 |
| --- | --- | --- |
| 首次按时还款 | +20 | 建立良好记录 |
| 每月按时还款 | +15 | 持续积累 |
| 连续3个月按时还款 | +30 | 额外奖励 |
| 连续6个月按时还款 | +50 | 大额奖励 |
| 提前还款（≥7天） | +25 | 鼓励提前 |
| 提前还清全部 | +40 | 一次性bonus |

#### 行为 2：募资行为
| 行为 | 得分 | 说明 |
| --- | --- | --- |
| 完成首轮募资（达目标） | +20 | 证明能力 |
| 完成第二轮募资 | +30 | 复购加分更多 |
| 第三轮及以后每次募资 | +40 | 老用户激励 |

#### 行为 3：生态贡献
| 行为 | 得分 | 说明 |
| --- | --- | --- |
| 邀请项目方入驻且成功募资 | +30 | 被邀请人完成首次还款后发放 |
| 参与平台治理投票 | +5 | 每次，月上限15分 |

### 1.5 升级示例：从新晋到成长 (200分 → 500分)
| 阶段 | 事件 | 分数变动 | 累计 |
| --- | --- | --- | --- |
| 起点 | 认证完成 | - | 200分 |
| 第1个月 | 首轮募资成功 | +20 | 220分 |
| 第2个月 | 首次按时还款 | +20 | 240分 |
| 第3个月 | 按时还款 + 连续3月bonus | +15+30 | 285分 |
| 第4-5个月 | 按时还款×2 | +30 | 315分 |
| 第6个月 | 按时还款 + 连续6月bonus | +15+50 | 380分 |
| 第7个月 | 邀请1个项目成功 | +30 | 410分 |
| 第8个月 | 第二轮募资成功 | +30 | 440分 |
| 第9个月 | 提前还款 | +25 | 465分 |
| 第10-11个月 | 按时还款×2 + 邀请1项目 | +60 | **525分 ✅** |

### 1.6 扣分规则（红线）
| 行为 | 扣分 | 后果 |
| --- | --- | --- |
| 还款逾期1-7天 | -30 | 延迟升级 |
| 还款逾期8-30天 | -80 | 可能掉回一档以下 |
| 还款逾期>30天 | -150 | 启动清算 |
| 断开数据连接 | -100 | 冻结新一轮募资权限 |
| 提交虚假数据 | -300 | 永久封禁，抵押品清算 |

## 2. 👤 投资者信用分系统 (Personal / Investors)

### 2.1 基础规则
| 项目 | 说明 |
| --- | --- |
| 连接钱包 | 100 分（起点） |
| 投资门槛 | 无，100分即可投资 |
| 信用分作用 | 等级权益 + 排名展示 + 治理权限 |

### 2.2 认购赚分
| 行为 | 信用分奖励 | 说明 |
| --- | --- | --- |
| 首次投资 | +30 | 破冰奖励 |
| 早鸟认购（项目开放24h内） | +20 | 抢先支持 |
| 大额认购（单笔≥$5,000） | +15 | 大额奖励 |
| 满额冲刺（项目进度>80%后认购） | +10 | 助力收官 |

### 2.3 持有赚分
| 持有行为 | 信用分奖励 | 说明 |
| --- | --- | --- |
| 持有满30天 | +10 | 满月奖励 |
| 持有满90天 | +25 | 季度忠诚 |
| 持有满180天 | +40 | 半年忠诚 |
| 持有满360天 | +80 | 年度忠诚 |
| 从未提前退出（自然年度） | +50 | 长期主义者徽章 |

### 2.4 复投赚分
| 复投行为 | 信用分奖励 | 说明 |
| --- | --- | --- |
| 首次复投（本息再投） | +20 | 资金不留平台 |
| 连续第2次复投 | +35 | 持续信任 |
| 连续第3次及以上复投 | +50 | 核心用户 |

### 2.5 核心权益体系
| 档位 | 信用分 | 核心权益 |
| --- | --- | --- |
| 探索者 | 100-499 | 标准认购时间，手续费 2% |
| 信徒 | 500-999 | 提前 6h 认购，手续费 1%，热门项目保证额度 |
| 传奇 | 1000+ | 提前 24h 认购，手续费 0.5%，专属额度预留 + 深度数据 |

**具体特权对比如下：**
| 权益 | 探索者 | 信徒 | 传奇 |
| --- | --- | --- | --- |
| 认购时间 | 公募开放时 | 提前 6 小时 | 提前 24 小时 |
| 手续费 | 2% | 1% | 0.5% |
| 热门项目额度 | 抢额度，可能售罄 | 保证基础额度（如$5,000） | 专属预留额度（如$50,000） |
| 项目数据 | 基础信息 | 详细财务 | 完整尽调报告 + 创始人连线 |

## 3. ⚖️ 治理系统 (Governance System)
> **核心原则**：投资者主导治理，项目方不参与投票

### 3.1 治理等级与门槛
低门槛设计：首次投资（+30）+ 早鸟认购（+20）= 150分，即可获得投票权！

| 信用分 | 治理等级 | 基础权限 |
| --- | --- | --- |
| 100-149 | 观察员 | 浏览提案，参与社区讨论，无投票权 |
| 150-499 | 参与者 | 参与基础类投票（平台规则、通用事项） |
| 500-999 | 治理者 | 参与所有投票 + 评论提案 |
| 1000+ | 提案者 | 发起新提案 + 参与所有投票 |

### 3.2 治理事项与权限矩阵
| 治理事项 | 观察员 | 参与者 | 治理者 | 提案者 | 特殊规则 |
| --- | --- | --- | --- | --- | --- |
| 平台规则变更（手续费、抵押率） | 👁️ | ✅ | ✅ | 可发起 | 全体投票 |
| 新项目白名单审核 | 👁️ | ✅ | ✅ | 可发起 | 全体投票 |
| 待清算项目处理（延期/重组） | 👁️ | ❌ | ✅ | 可发起 | 仅投资者可投 |
| 立即清算触发 | 👁️ | ❌ | ✅ | 可发起 | 仅投资者可投 |
| 争议仲裁 | 👁️ | ❌ | ✅ | 可发起 | 利益相关方权重×2 |
| 平台发展方向（新功能优先级） | 👁️ | ✅ | ✅ | 可发起 | 全体投票 |
| 紧急暂停（系统风险） | 👁️ | ❌ | ✅ | 可发起 | 快速通道，24h决 |

### 3.3 投票权重计算
公式：**权重 = 信用分 × 角色系数 × 利益系数**

| 系数类型 | 情况 | 数值 |
| --- | --- | --- |
| **角色系数** | 投资者 | 1.0 |
| | 项目方（已认证） | 0.8 |
| | 项目方（蓝筹等级） | 1.0 |
| **利益系数** | 与该事项无直接利益 | 1.0 |
| | 持有该项目份额（投资者） | 1.5 |
| | 是该项目的项目方 | 0（禁止投票） |
| | 被仲裁的项目方 | 0（禁止投票） |

### 3.4 治理奖励机制
参与治理可反向激励更多信用分：

| 行为 | 奖励分数 | 月上限 | 说明 |
| --- | --- | --- | --- |
| 参与投票 | +5 | 25分 | 每次有效投票 |
| 投票与最终结果一致 | +10 | 30分 | 明智决策奖励 |
| 评论提案被点赞 TOP3 | +15 | 15分 | 优质观点 |
| 发起提案被通过（提案者专属） | +30 | 30分 | 建设性贡献 |
| 参与紧急治理 | +20 | 20分 | 快速响应奖励 |
